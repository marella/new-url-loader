This example uses `@svgr/webpack` with `new-url-loader` instead of `url-loader`. See [`webpack.config.js`](webpack.config.js), [`src/index.js`](src/index.js) and [`src/index.css`](src/index.css).

**Install**

```sh
npm install
```

**Build**

```sh
npm run build
```

**Serve**

```sh
npm run serve
```
